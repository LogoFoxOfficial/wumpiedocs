# Skip To

{% hint style="info" %}
You can see the numbers in the queue with [**W!queue**](queue.md)\*\*\*\*
{% endhint %}

#### Description:

Skips songs to song's position that you give.

#### Usage:

```text
W!skipto {Number}
```

**Arguments:**

```text
{Number} - A number of song in queue.
```

**Syntax:**

```text
{} - Required.
[] - Optional.
```
