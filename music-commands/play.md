# Play

{% hint style="warning" %}
You can't play Spotify songs with this command!\
Use [W!spotify](spotify.md) to play Spotify songs/playlists.
{% endhint %}

#### Description:

Play music from youtube.

#### Usage:

```
W!play {Song Title/Link}
```

**Arguments:**

```
{Song Title/Link} - A song title or a link.
```

**Syntax:**

```
{} - Required.
[] - Optional.
```
