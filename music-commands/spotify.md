# Spotify

{% hint style="info" %}
Links can be Playlists and Songs (https://open.spotify.com/track/example).
{% endhint %}

#### Description:

Plays a song/playlist from the Spotify link.

#### Usage:

```
W!spotify {Link}
```

**Arguments:**

```
{Link} - A link of spotify music/playlist.
```

**Syntax:**

```
{} - Required.
[] - Optional.
```
