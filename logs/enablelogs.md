# Enablelogs

**Description:**

Enables message, invite, role, kick and ban-log

#### Usage:

```text
W!Enablelogs {Channel Mention}
```

#### Permissions:

```text
For this command you need Admin permission
```

#### Arguments:

```text
{Channel Mention} - A channel mention or ID
```

#### Syntax:

```text
{} - Required.
[] - Optional
```
