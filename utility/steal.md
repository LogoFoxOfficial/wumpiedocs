# Steal

**Description:**

Steals/Copys a emoji from a server&#x20;

{% hint style="warning" %}
Nitro is required for this command
{% endhint %}

#### Usage:

```
W!steal {Emoji} {Name} 
```

#### Permissions:

```
For this command you need Admin permission
```

#### Arguments:

```
{Emoji} - emoji from other server 
{Name} - name of emoji that will be on your server
```

#### Syntax:

```
{} - Required.
[] - Optional.
```
