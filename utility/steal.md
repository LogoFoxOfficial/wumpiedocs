# Steal

**Description:**

Steals/Copys a emoji from a server 

{% hint style="warning" %}
Nitro is required for this command
{% endhint %}

#### Usage:

```text
W!steal {Emoji} {Name} 
```

#### Permissions:

```text
For this command you need Admin permission
```

#### Arguments:

```text
{Emoji} - emoji from other server 
{Name} - name of emoji that will be on your server
```

#### Syntax:

```text
{} - Required.
[] - Optional.
```
