# Translate

**Description:**

Translate message 

#### Usage:

```text
w!translate {Message} {language} 
```

#### Permissions:

```text
No special permissions needed for this command
```

#### Arguments:

```text
{Message} - message that you want to translate
{language} - for example: English -> en 
```

#### Syntax:

```text
{} - Required.
[] - Optional.
```
