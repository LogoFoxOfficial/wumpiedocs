# Rank

**Description:**

Shows you your current level, how much xp you have at your current level and how much is needed for a level-up

#### Usage:

```
W!rank [Mention]
```

#### Permissions:

```
No special permissions needed for this command
```

#### Arguments:

```
[Mention] - Optional user mention or User ID.
```

#### Syntax:

```
{} - Required.
[] - Optional.
```
