# Setlevelchannel

**Description:**

Sets the channel where the leveling messages will be send

#### Usage:

```
W!setlevelchannel {Channel Mention}
```

#### Permissions:

```
For this command you need Admin permission
```

#### Arguments:

```
{Channel Mention} - A channel mention or ID
```

#### Syntax:

```
{} - Required.
[] - Optional.
```
