# Enableantilink

**Description:**

Enables the antilink-system from wumpie. He will delete discord.gg and discord.com links immediately

#### Usage:

```text
W!enableantilink
```

#### Permissions:

```text
For this command you need Admin permission
```

#### Arguments:

```text
None
```

#### Syntax:

```text
{} - Required.
[] - Optional.
```
