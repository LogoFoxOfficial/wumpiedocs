# Mute

#### Description:

Mute a naughty person who breaks the rules.

#### Usage:

```text
W!mute {Mention} {Reason}
```

#### Permission:

```text
You need Manage Roles permission to use this command.
```

**Arguments:**

```text
{Mention} - A user mention or User ID.
{Reason} - A reason for mute.
```

**Syntax:**

```text
{} - Required.
[] - Optional.
```
