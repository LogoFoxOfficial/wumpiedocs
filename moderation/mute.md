# Timeout

#### Description:

Timeout a naughty person who breaks the rules.

#### Usage:

```
W!timeout {Mention|ID|Username} {Time}
```

#### Permission:

```
You need kick permissions to use this command.
```

**Arguments:**

```
{Mention} - A user mention or User ID.
{Time} - Values: 60s, 300s, 600s, 1h, 1d, 1w
```

**Syntax:**

```
{} - Required.
[] - Optional.
```
