# Giverole

#### Description:

Gives a role you tagged to an user you tagged

#### Usage:

```
W!giverole {Mention} {Role Mention}
```

#### Permissions:

```
For this command you need Manage Roles permission
```

#### Arguments:

```
{Mention} - A user mention or User ID.
{Role Mention} - A role mention or role ID
```

#### Syntax:

```
{} - Required.
[] - Optional.
```
