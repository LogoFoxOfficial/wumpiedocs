# Setmainrole

#### Description:

You can specify the mainrole of your server, highly recommended when you want to use lockdown

#### Usage:

```text
W!Setmainrole {Role Mention}
```

#### Permissions:

```text
For this command you need Admin permission
```

#### Arguments:

```text
{Role Mention} - A role mention or role ID
```

#### Syntax:

```text
{} - Required.
[] - Optional.
```
