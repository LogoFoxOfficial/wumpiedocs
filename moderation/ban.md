# Ban

#### Description:

Bans a member from the server you've executed the command.

#### Usage:

```text
W!ban {Mention/ID} [Reason]
```

#### Permission:

```text
You need Ban Members permission to use this command.
```

#### Arguments:

```text
{Mention} - A user mention or User ID.
[Reason] - A reason for ban.
```

**Syntax:**

```text
{} - Required.
[] - Optional.
```
