# Warn

#### Description:

Warn naughty users for rule-breaking.

#### Usage:

```
W!warn {Mention} {Reason}
```

#### Permission:

```
You need Kick Members permission to use this command.
```

**Arguments:**

```
{Mention} - A user mention.
{Reason} - A reason for warn.
```

**Syntax:**

```
{} - Required.
[] - Optional.
```
