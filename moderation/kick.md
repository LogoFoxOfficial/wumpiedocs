# Kick

#### Description:

Kicks a member from the server you've executed the command.

#### Usage:

```
W!kick {Mention/ID} [Reason]
```

#### Permission:

```
You need Kick Members permission to use this command.
```

**Arguments:**

```
{Mention} - A user mention or User ID.
[Reason] - A reason for kick.
```

**Syntax:**

```
{} - Required.
[] - Optional.
```
