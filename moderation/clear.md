# Clear

#### Description:

Deletes the specified amount of messages in a channel  


#### Usage:

```text
W!clear/purge {Amount}
```

#### Permissions:

```text
For this command you need Manage Messages permission
```

#### Arguments:

```text
{Amount} - The amount of messages you want to delete
```

#### Syntax:

```text
{} - Required.
[] - Optional.
```
